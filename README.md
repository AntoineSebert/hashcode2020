## Toolchain

### Python

Install the interpreter for Python 3.8:
```
sudo apt-get install python3.8
```

You can then check the version:
```
python --version
```

Or alternatively:
```
python3 --version
```

### Pip

Install the package manager for Python:
```
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
```
```
python get-pip.py
```
